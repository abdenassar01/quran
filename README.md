## setup

- clone this repository

```
git clone https://github.com/abdenassar01/quran.git
```

- install dependancies

_using yarn:_
```
yarn 
```

_using npm:_
```
npm install
```

- run locally

**yarn**

```
yarn run dev   
```


**npm**

```
npm run dev   
```
